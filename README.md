# 智绘王屋 - 当地好物展示站

类似博物馆风格的单页展示站，支持**跳转淘宝店铺**（当前为占位）和 **AI 智能咨询**（预算、房间、游玩安排等）。

## 使用方式

- 用浏览器直接打开 `index.html` 即可本地预览。
- 或使用本地服务器（如 VS Code Live Server、`npx serve .`）获得更好体验。

## 淘宝店铺跳转

- **当前**：所有「进入淘宝店铺」点击会弹出提示弹窗（店铺筹备中），并提供一个「先逛淘宝」的通用链接。
- **店铺建好后**：在 `app.js` 中搜索 `a[href="#taobao"]`，可改为直接跳转：把相关链接的 `href="#taobao"` 改成您的店铺地址（如 `https://shop.taobao.com/shop/xxxxx`），并删除或注释掉「打开淘宝弹窗」的逻辑（`openTaobaoModal` 相关代码）。

或在 `index.html` 中把「进入淘宝店铺」的链接统一改为您的店铺 URL，并移除/修改弹窗相关 HTML 与 JS。

## AI 智能咨询

- 点击导航栏「AI 智能咨询」、快捷入口「AI 智能咨询」或右下角悬浮按钮可打开聊天窗口。
- AI 会根据关键词回答：**预算**（几人几天花费）、**房间/住宿**、**游玩安排/行程**、**当地/王屋/美食**等。
- 当前为本地规则回复，无需 API。后续可接入 OpenAI/通义等：在 `app.js` 的 `getAiReply` 处改为调用您的接口即可。

## 文件说明

- `index.html` - 主页面
- `styles.css` - 样式（暖色、文化感）
- `app.js` - 轮播、淘宝弹窗、AI 对话逻辑
